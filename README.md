<h3>1.INTRODUÇÃO</h3></br>
 📍 Este trabalho tem como objetivo aplicar os conceitos básicos da linguagem C por meio da resolução de exercícios práticos, permitindo o desenvolvimento do raciocínio lógico e da capacidade de solucionar problemas computacionais.
<hr>
<h3>2.DESCRIÇÃO DO PROBLEMA</h3>

O presente trabalho consiste na implementação de uma série de exercícios em linguagem C, abordando diferentes situações práticas relacionadas à programação. Os problemas propostos envolvem:
<ul>
	<li>entrada e saída de dados pelo usuário;</li> 
	<li>operações matemáticas básicas;</li> 
	<li>estruturas condicionais para tomada de decisão;</li> 
	<li>estruturas de repetição para execução de tarefas iterativas;</li> 
	<li>manipulação de sequências numéricas e vetores.</li>
</ul>  
Cada exercício foi desenvolvido com o objetivo de consolidar o entendimento dos conceitos fundamentais da programação, além de estimular a organização e a clareza na escrita do código.
<hr>

<h3>3.APRENDIZAGEM OBTIDA</h3>

 📍 A realização dos exercícios permitiu desenvolver habilidades em lógica de programação, uso de estruturas condicionais e de repetição, além da manipulação de dados em linguagem C.
